# CardioRisk AI: Heart Disease Prediction

## Description
Built a machine learning model to predict cardiovascular disease using Python and Scikit-learn.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Model
- Logistic Regression

## Accuracy
- 72%

## Features
- Age, Blood Pressure, Cholesterol, BMI, etc.

## Conclusion
Logistic Regression performed better than Decision Tree for this dataset.
